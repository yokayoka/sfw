位置情報の共有

位置の情報を言葉だけで伝えるのは大変やっかいです。街中であれば、そこのコンビニを右手に曲がって、などと、お互いが知っている建物などの地物を手がかりに伝えることも、ある程度は可能ですが、経路が複雑になれば伝達が難しくなります。まして、山であれば、任意の場所を言葉だけで伝えるのは不可能に近いでしょう。GPSを使えば緯度経度から正確に場所を伝えることは出来ますが、緯度経度の数字から場所をイメージすることは常人には不可能でしょう。場所を伝えるには、やはり、地図が一番便利なのですが、毎回地図の画像をメールやSNSに添付するのも大変です。Google Mapなどでは、kmlやkmzという地理情報を記述したファイルを読み込むことは出来ますが、ちょっとした手間がかかります。SNSで気に入ったサイトの情報をやりとりするように手軽な形で、場所の情報をシェアできないかと考えてこのサイトを作成しました。
